Alita Robot is a powerful Telegram group management bot built with Go and the Gotgbot library. It provides comprehensive moderation tools, spam protection, customizable greetings, keyword filters, notes, and much more — all running at Go-native speed with Redis caching and PostgreSQL persistence.

## Get started in minutes

<Steps>
  <Step title="Clone and configure">
    Clone the repository and copy `sample.env` to `.env`. Set your `BOT_TOKEN`, `OWNER_ID`, and `MESSAGE_DUMP` values.
  </Step>
  <Step title="Start with Docker Compose">
    Run `docker-compose up -d` to start Alita along with PostgreSQL and Redis. Database migrations run automatically.
  </Step>
  <Step title="Add your bot to a group">
    Open Telegram, add your bot to a group, and promote it to admin. Alita is ready to manage your community.
  </Step>
  <Step title="Configure features">
    Use commands like `/setwelcome`, `/setflood`, and `/blacklist` to customize Alita for your group's needs.
  </Step>
</Steps>
